# Java-NumberArray

A simple Java program that creates and prints a number array based on user input.

## Description

This project contains a basic Java console application that prompts the user to enter a number, initializes an array with values from 0 to the entered number minus one, and prints each value to the console. It demonstrates fundamental Java programming concepts including input handling, arrays, and loops.
